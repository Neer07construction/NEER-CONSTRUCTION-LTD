Setup Guide
Connect your WhatsApp Business account — step by step
🚀 How to Go Live in 4 Steps
1
Get a Meta Developer Account & WhatsApp API Access
Go to developers.facebook.com → Create an App → Choose "Business" → Add the WhatsApp product.

You will get:
• Access Token (WHATSAPP_TOKEN)
• Phone Number ID (WHATSAPP_PHONE_ID)
2
Get Your Claude (Anthropic) API Key
Go to console.anthropic.com → Sign up → API Keys → Create Key.
This is your ANTHROPIC_API_KEY.
3
Upload files to your server & install
Upload all files to your server, then run:
npm install cp .env.example .env nano .env # fill in your 4 keys node server.js
4
Connect webhook in Meta Developer Console
In Meta Console → WhatsApp → Configuration → Webhook:

• Callback URL: http://YOUR-SERVER-IP:3000/webhook
• Verify Token: same value as your VERIFY_TOKEN in .env
• Subscribe to: messages

✅ Done! Your bot is live.
⚙️ Your .env File (fill these in)
WHATSAPP_TOKEN=your_whatsapp_access_token_here WHATSAPP_PHONE_ID=your_phone_number_id_here VERIFY_TOKEN=NeerBotSecret2024 ANTHROPIC_API_KEY=sk-ant-your-key-here PORT=3000
